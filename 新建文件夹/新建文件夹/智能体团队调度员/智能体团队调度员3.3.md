## 四、多轮对话上下文保持

### 4.1 核心机制

多轮对话时，自动在消息中携带对话历史，确保上下文连贯。

### 4.2 对话历史格式

```json
{
  "conversation_id": "conv_xxx",
  "turn": 5,
  "history": [
    {"role": "user", "content": "第一轮问题"},
    {"role": "assistant", "content": "第一轮回答"}
  ]
}
```

### 4.3 上下文保持规则

| 规则 | 说明 |
|------|------|
| 自动携带 | 每条消息自动携带最近N轮对话历史 |
| 上下文窗口 | 默认保留最近10轮对话（约2000token） |
| 摘要压缩 | 超过窗口时自动摘要早期对话 |
| 上下文继承 | 新任务自动继承当前对话上下文 |

## 七、固定团队机制

3个预设团队，有明确分工和工作流程。

### 7.1 开发团队 (dev_team)

- **负责人**: dispatcher_agent
- **角色**: 项目经理、需求分析、架构设计、代码开发、测试验证、部署发布
- **流程**: 需求分析→架构设计→代码开发(可并行)→测试验证→部署发布
- **适用**: 软件项目开发

### 7.2 研究团队 (research_team)

- **负责人**: society_of_mind_agent
- **角色**: 团队协调、调研收集、语义分析、知识图谱、报告撰写、质量审核
- **流程**: 调研收集→语义分析→知识关联→多视角分析→报告撰写(可并行)
- **适用**: 研究、调研、分析

### 7.3 审核团队 (review_team)

- **负责人**: monitor_agent
- **角色**: 审核监督、内容审核、质量审核、最终审核
- **流程**: 初审→复审→终审→监督报告
- **适用**: 内容审核、质量把关

| 类型 | 说明 | 发送方 |

  "content": {"title": "...", "body": "..."},

## 十五、自主学习能力

系统支持基于经验的自主学习，核心机制：
- **情景记忆积累**：每次任务执行后自动记录到情景记忆
- **相似任务检索**：通过`get_similar_tasks`接口查询历史相似任务

### 15.1 学习流程

新任务→查询相似→分析案例→优化选择→执行→记录结果

## 十七、执行流程

```
用户输入→Step1:接收任务→Step2:图谱查询+上下文分析→Step3:工作流选择→Step4:并行调度→Step5:结果聚合→Step6:返回结果
```

**Step1**：解析输入、关联/创建上下文、检查相似任务
**Step2**：分词查询、分析偏好、确定博弈模式
**Step3**：选择工作流模板（默认parallel_execution）
**Step4**：总线分发、启动监控、记录历史
**Step5**：收集输出、整合监控报告、更新记忆
**Step6**：输出结果、记录日志

## 十八、插件化智能体系统

### 18.1 核心接口

```python
class IAgent(ABC):
    @abstractmethod
    def get_id(self) -> str: pass
    @abstractmethod
    def get_metadata(self) -> Dict[str, Any]: pass
    @abstractmethod
    async def execute(self, task, context) -> Dict[str, Any]: pass
    @abstractmethod
    async def health_check(self) -> bool: pass
```

### 18.2 动态智能体发现器

```python
class AgentDiscovery:
    async def scan_agents(self) -> List[Dict[str, Any]]:
        agents = []
        for scan_config in self.config.get("scan_config", {}).get("directories", []):
            directory = Path(scan_config["path"])
            pattern = scan_config.get("pattern", "*_agent.md")
            if not directory.exists(): continue
            files = list(directory.rglob(pattern)) if scan_config.get("recursive", True) else list(directory.glob(pattern))
            for file_path in files:
                if not self._should_exclude(file_path):
                    agent_info = self._parse_agent_file(file_path)
                    if agent_info: agents.append(agent_info)
        return agents
```
