## 规则解释流模板详解

### 模板ID: rule_interpreter_flow

**模式类型**: 规则解释流

**核心功能**: 解析自然语言规则文档，转换为结构化执行计划

**触发条件**: 
- 任务包含"规则"、"解释"、"文档"、"逻辑"、"转换"、"说明"等关键词

**执行流程**:
1. **解析意图**: 识别用户需求类型（规则查询、规则转换、规则验证）
2. **读取文档**: 从上下文中获取规则文档内容
3. **逻辑转换**: 将自然语言规则转换为结构化表示
4. **生成计划**: 根据任务需求制定执行计划
5. **返回结果**: 输出结构化解析结果和执行建议

**输出格式**:
```json
{
  "status": "success/error/partial",
  "data": {
    "parsed_rules": [...],
    "execution_plan": [...],
    "dependencies": [...]
  },
  "message": "说明文字"
}
```

**关联智能体**: rule_interpreter_agent
