```python
class VectorMatcher:
    def cosine_similarity(v1, v2):
        dot = sum(a*b for a,b in zip(v1,v2))
        n1, n2 = math.sqrt(sum(a*a for a in v1)), math.sqrt(sum(b*b for b in v2))
        return dot / (n1*n2) if n1 and n2 else 0.0
```

## 二十三、错误处理

| 错误码 | 说明 | 处理 |
|-------|-----|-----|
| ERR_AGENT_UNAVAILABLE | 智能体不可用 | 切换备用 |
| ERR_INTENT_RECOGNITION | 意图识别失败 | 回退通用路由 |
| ERR_KG_QUERY | 图谱查询失败 | 使用向量相似度 |
| ERR_TIMEOUT | 执行超时 | 重试/终止 |
| ERR_CIRCUIT_BREAKER | 断路器打开 | 降级响应 |
