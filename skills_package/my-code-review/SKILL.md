# 代码审查技能

## 工作流程

1. **代码收集**：识别并收集需要审查的代码文件
2. **静态分析**：检查代码结构、语法和潜在问题
3. **问题识别**：识别潜在的 bug、安全问题和性能问题
4. **改进建议**：提供具体的改进建议和最佳实践

## 触发词

- 代码审查
- 代码质量
- 代码分析
- 代码评估
- 代码检查
- 代码改进
- 代码优化
