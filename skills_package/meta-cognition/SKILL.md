# Meta-Cognition Skill

## 概述

元认知技能，用于在每次任务执行前后自动记录和分析调度策略的有效性。支持三种博弈模式和智能推荐功能，提供完整的任务执行审计和统计分析。

**v2.8.1 更新**：新增守护进程模式，支持全自动触发！

## 核心功能

### 1. 前置钩子 (Pre-Task Hook)

在任务执行前自动触发，记录任务调度前的状态并做出调度决策。

**功能特性**：
- 任务类型自动识别
- 调度模式智能选择（90%+ 触发率）
- 敏感信息自动过滤
- 智能体推荐

### 2. 后置钩子 (Post-Task Hook)

在任务执行后自动触发，记录执行结果并生成统计信息。

## 三种博弈模式

### 模式1：辩论模式（串行模拟并行）

**触发关键词**：辩论、对比、优缺点、风险评估、多角度分析

### 模式2：降维打击模式（流水线博弈）

**触发关键词**：优化、改进、提升、增强、完善、重构

### 模式3：深度设计模式（三段式协同）

**触发关键词**：设计、架构、实现、搭建、创建、开发
