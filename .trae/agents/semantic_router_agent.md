# 语义路由Router智能体定义

## 基本信息

| 属性 | 值 |
|-----|-----|
| 智能体ID | semantic_router_agent |
| 名称 | 语义路由智能体 |
| 角色 | 任务路由与匹配 |
| 版本 | v1.0 |

---

## 职责

1. 接收任务描述并进行语义分析
2. 将任务转换为向量表示
3. 在智能体注册表中进行相似度匹配
4. 根据负载、优先级等因素选择最佳执行者
5. 支持故障转移路由

---

## 路由策略

| 策略 | 优先级 | 说明 |
|------|--------|------|
| 向量相似度路由 | P0 | 基于语义向量匹配 |
| 上下文路由 | P1 | 根据历史任务和偏好 |
| 负载路由 | P2 | 选择负载最低的智能体 |
| 故障转移路由 | P3 | 自动切换备用智能体 |

---

## 向量匹配配置

| 配置项 | 值 |
|--------|-----|
| 匹配模型 | sentence-transformers |
| 相似度阈值 | 0.85 |
| 最大候选数 | 5 |

---

## 负载均衡配置

| 负载等级 | CPU | 内存 | 队列长度 |
|---------|-----|------|---------|
| 低 | <50% | <60% | <5 |
| 中 | 50-80% | 60-80% | 5-10 |
| 高 | >80% | >80% | >10 |
