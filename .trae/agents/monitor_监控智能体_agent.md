## 自动化闭环流程

```
1. 规则加载
   ↓
2. 创建共用上下文
   ↓
3. 并行启动（执行智能体 + 监控智能体同步）
   ↓
4. 采集智能体行为
   ↓
5. 规则匹配 + 违规判定
   ↓
6. 自动处理（警告/修复/阻断）
   ↓
7. 记录日志 + 生成报告
   ↓
8. 反馈学习 → 返回步骤1继续循环
```

## 共用上下文机制

### 上下文对象结构

```python
class Context:
    context_id: str                    # 上下文唯一标识
    data: Dict[str, Any]              # 通用数据存储
    agent_state: Dict[str, Any]       # 智能体状态
    tool_calls: List[Dict]            # 工具调用记录
    memory_state: Dict[str, Any]      # 记忆系统状态
    messages: List[Dict]              # 对话消息历史
    created_at: str                   # 创建时间
    last_updated: str                 # 最后更新时间
```

### 使用方式

1. **创建上下文**（执行智能体和监控智能体共用）
   ```python
   context = monitor.create_context("my_task_001")
   ```

2. **执行智能体记录行为**
   ```python
   context.add_message("user", "用户输入")
   context.add_tool_call("tool_name", params, result)
   ```

3. **监控智能体同步检查**
   ```python
   report = monitor.monitor("my_task_001", agent_behavior)
   ```

## 使用示例

### Python 代码使用

```python
from monitor_system import MonitorAgentV4

# 初始化监控智能体（自动加载 L1/L2/L3 所有规则）
monitor = MonitorAgentV4(rules_path=".trae/rules")

# 创建共用上下文
context = monitor.create_context("my_project_context_001")

# 记录智能体行为（执行智能体和监控智能体共用此上下文）
context.add_message("user", "帮我搜索一下如何学习Python")
context.add_tool_call("search_web", {"query": "Python学习"}, {"result": "..."})

# 执行监控（与执行智能体同步）
report = monitor.monitor("my_project_context_001", {
    "agent_name": "code_executor_agent",
    "task_completed": True
})

# 查看监控报告
print(f"规则总数: {report['rules_summary']['total']}")
print(f"违规数量: {len(report['violations'])}")
print(f"警告数: {report['process_result']['warnings_issued']}")
```
