## 使用方式

当你说以下关键词时，对应的Skill会自动激活：

```
"帮我调试这个错误" → 激活 auto-debug
"设置一个hook" → 激活 auto-hook
"生成README" → 激活 auto-doc
"重构这个项目" → 激活 auto-refactor
"读取本地笔记" → 激活 local-privacy
```
