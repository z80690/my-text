## 一、博弈模式触发词

| 模式名称 | 触发关键词 | 执行策略 |
|---------|-----------|---------|
| **优化改进** | 优化、改进、重构、审查、提升、增强 | 生成 → 挑剔 → 融合 |
| **深度设计** | 设计、架构、方案、对比、规划、蓝图 | 蓝图 → 挑战 → 融合 |
| **辩论模式** | 辩论、权衡、优缺点、利弊、对比分析 | 视角A → 视角B → 裁决 |

## 六、触发优先级

| 优先级 | 类别 | 说明 |
|-------|------|-----|
| P0 | 系统级触发词 | 版本切换、系统控制 |
| P1 | 博弈模式触发词 | 决定执行策略 |
| P2 | 工作流模板触发词 | 选择工作流 |
| P3 | 智能体触发词 | 选择执行智能体 |
| P4 | 技能库触发词 | 选择辅助技能 |

## 七、触发示例

### 示例1：优化任务
```
"审查并优化项目根目录下的 main.py 文件，确保其符合PEP 8规范"
→ 触发：博弈模式-优化改进 → 工作流-监控流 → 智能体-代码执行智能体
```

### 示例2：设计任务
```
"设计一个企业级智能体框架的架构方案"
→ 触发：博弈模式-深度设计 → 工作流-博弈模式 → 智能体-心智社会智能体
```

### 示例3：规则解释任务
```
"解释这份规则文档的逻辑结构"
→ 触发：工作流-规则解释流 → 智能体-规则解释智能体
```
