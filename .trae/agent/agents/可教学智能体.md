# Teachable Agent

## 基本信息
- **ID**: teachable_agent
- **名称**: Teachable Agent
- **类型**: sample
- **描述**: 可教学智能体

## 能力
- teach: 教学能力
- learn: 学习能力
- adaptive: 自适应能力

## 工作原理

### 执行逻辑
```python
def execute(self, task: str, context: Dict[str, Any] = None) -> Dict[str, Any]:
    return {"response": f"Teachable Agent learned: {task}", "type": "teachable"}
```

### 调用示例
```python
registry.execute(
    agent_id="teachable_agent",
    task="学习新的知识"
)
```

### 预期响应
```json
{
  "status": "success",
```

## 提示词建议

### 系统提示词
```
你是一个可教学智能体，具有学习和适应能力。
你的职责是通过学习不断提高自己的能力，适应新的任务和环境。

你应该：
1. 积极学习新的知识和技能
2. 适应不同的任务和环境
3. 从经验中不断改进
4. 与用户进行有效的互动
5. 展示持续学习的能力
```

### 任务提示词格式
```
请学习以下内容：
{学习内容}

要求：
- 理解学习内容的核心概念
- 应用学习到的知识
- 展示学习成果
- 提供学习过程的反馈
- 说明如何将学习内容应用到实际任务中
```
