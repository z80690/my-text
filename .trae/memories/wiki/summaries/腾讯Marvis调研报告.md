# 腾讯 Marvis（贾维斯）调研报告

## 核心信息
- **创建日期**: 2026-05-23
- **最后更新**: 2026-05-23
- **来源**: 项目根目录调研报告

## 核心内容

### 产品概述
腾讯 Marvis 是一款操作系统层级的个人 AI 助手，昵称"腾讯版贾维斯"。

**核心定位**: "用自然语言操控整台电脑/手机的系统级智能管家"

**产品口号**: "一句话让电脑自己干活"

### 核心功能
1. **深度系统操作**: 硬件检测、系统设置、性能优化、软件管理
2. **智能文件管理**: 内容搜索、文档解析、智能分类、格式转换
3. **办公自动化**: 文档生成、数据分析、资料整理、翻译服务
4. **跨端协同**: 手机控电脑、远程解锁、电脑控手机、文件传递

### 技术架构
**多 Agent 协作体系（6个 Agent）**:
- 主 Agent（项目经理）
- File Agent（文件管家）
- Computer Agent（系统运维）
- App Agent（应用操控）
- Browser Agent（网页交互）
- Search Agent（信息检索）

### 运行模式
- **效率模式**: 端云协同，复杂任务借助云端模型
- **隐私模式**: 完全离线，基于 Qwen 端侧模型，数据不出设备

### 安全机制
- L2 级安全兜底机制
- 敏感操作需用户确认
- 支付动作必须由用户本人完成

### 支持平台
- Windows 10/11（已开放）
- macOS（已开放）
- Android（已开放）
- iOS（开发中，预计 2026 年 6 月中旬）

### 系统要求
- CPU: 8 核（Intel Core i7 或 AMD Ryzen 7）
- 内存: 16GB RAM
- 存储: SSD，2GB 可用空间
- 显卡: 推荐独立显卡

### 免费额度
- 每日免费 Token: 1000 万 Token/天/用户
- 获取方式: 官网直接下载，无需邀请码

### 产品对比
| 维度 | 传统 AI 助手 | Marvis |
|------|-------------|--------|
| 定位 | 通用对话型 | 操作系统层级 |
| 权限 | 应用层（挂在系统上） | 系统层（长在系统里） |
| 文件操作 | 需手动上传 | 直接访问本地文件 |
| 系统控制 | 有限（模拟点击） | 深度操控（底层 API） |
| 跨端能力 | 弱 | 强（手机控电脑、电脑控手机） |

### 行业意义
代表 AI 助手发展的新方向：
- 从"聊天工具"向"系统管家"升级
- 从"云端对话"向"终端执行"演进
- 从"单一 Agent"向"多 Agent 协作"发展
