## 规则架构

```
L1 (宪法) - 根本原则，不可逾越
    ↓
L2 (基本法) - 协作规范，必须遵守
    ↓
L3 (具体规范) - 执行指南，具体操作
```

### 规则优先级

> **官方标准**: User input > Custom Agent prompts > `user_rules.md` > `project_rules.md`

| 优先级 | 来源 | 说明 |
|--------|------|------|
| P0 | User input | 用户直接输入 |
| P1 | Custom Agent prompts | 自定义智能体提示词 |
| P2 | user_rules.md | 个人规则 |
| P3 | project_rules.md | 项目规则 |

### 规则类型优先级（质量为最高优先级）
1. **质量规则** > **安全规则** > **协作规则** > **工程规则**
2. **L1** > **L2** > **L3**
