## 五、博弈决策

### 5.1 决策方法选择

| 场景 | 方法 | 说明 |
|------|------|------|
| 3个以上来源，多数一致 | **投票机制** | 取多数意见 |
| 风险敏感场景 | **Minimax** | 最坏情况最优 |
| 多方博弈、无明显多数 | **纳什均衡** | 稳定策略 |

### 5.2 投票机制实现

```python
def vote(answers: List[str]) -> str:
    """简单多数投票"""
    from collections import Counter
    counts = Counter(answers)
    return counts.most_common(1)[0][0]
```

### 5.3 Minimax实现

```python
def minimax_decision(options: List[Option], context: Context) -> Option:
    """
    Minimax决策：选择最坏情况下最好的选项
    """
    scores = []
    for option in options:
        worst_case = option.evaluate_worst_case(context)
        scores.append((option, worst_case))
    return max(scores, key=lambda x: x[1])[0]
```

### 5.4 纳什均衡实现

```python
def nash_equilibrium(payoff_matrix: List[List[float]]) -> Tuple[int, int]:
    """
    纳什均衡：找到稳定策略组合
    """
    best_row, best_col = 0, 0
    for i in range(len(payoff_matrix)):
        for j in range(len(payoff_matrix[0])):
            if payoff_matrix[i][j] > payoff_matrix[best_row][best_col]:
                best_row, best_col = i, j
    return best_row, best_col
```

## 六、置信度标注

### 6.1 置信度等级定义

| 等级 | 标注 | 颜色 | 说明 | 示例 |
|------|------|------|------|------|
| Level 1 | 🟢确定 | 绿 | 权威来源+多方验证一致 | 官方文档+多个来源确认 |
| Level 2 | 🟡可能 | 黄 | 单来源+无冲突 | 单个权威来源，无其他验证 |
| Level 3 | 🟠存疑 | 橙 | 多来源有分歧 | 2个以上来源结论不一致 |
| Level 4 | 🔴未知 | 红 | 无法验证 | 来源不可访问或信息不足 |

### 6.2 置信度计算规则

```python
def calculate_confidence(sources: List[Source], consistency: float) -> int:
    """
    计算置信度等级
    - sources: 来源列表
    - consistency: 一致性得分 (0-1)
    """
    avg_credibility = sum(s.credibility for s in sources) / len(sources)
    
    if consistency >= 0.9 and avg_credibility >= 80:
        return 1  # 🟢确定
    elif consistency >= 0.7 and avg_credibility >= 70:
        return 2  # 🟡可能
    elif consistency >= 0.5:
        return 3  # 🟠存疑
    else:
        return 4  # 🔴未知
```

```python
def semantic_match(query: str, candidates: List[str]) -> float:
    """
    语义相似度匹配
    - 返回最高相似度分数
    """
    query_vec = vectorize(query)
    best_score = 0.0
    for candidate in candidates:
        cand_vec = vectorize(candidate)
        score = cosine_similarity(query_vec, cand_vec)
        best_score = max(best_score, score)
    return best_score
```

### 7.3 自适应学习

```python
class VerificationLearner:
    """根据验证成功率自动调整策略"""
    
    def record_result(self, claim: str, verified: bool):
        """记录验证结果"""
        self.history.append({"claim": claim, "verified": verified})
    
    def get_success_rate(self) -> float:
        """获取成功率"""
        if not self.history:
            return 0.5
        return sum(1 for r in self.history if r["verified"]) / len(self.history)
    
    def should_adjust_strategy(self) -> bool:
        """判断是否需要调整策略"""
        return self.get_success_rate() < 0.6
```
