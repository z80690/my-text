## 1. 会话记忆模块（L3-R010）

### 1.1 核心能力

| 能力 | 说明 |
|-----|------|
| 上下文存储 | 当前对话的所有消息和工具调用 |
| TTL管理 | 自动过期清理 |
| 窗口限制 | 上下文窗口管理 |

### 1.2 数据结构

```python
@dataclass
class SessionContext:
    conversation_id: str
    turn: int
    messages: List[Message]
    created_at: datetime
    last_accessed: datetime
    ttl: int  # 秒
```

### 1.3 配置参数

| 参数 | 值 | 说明 |
|-----|-----|------|
| window_size | 10 | 上下文窗口大小（轮数） |
| max_tokens | 2000 | 最大Token数 |
| auto_summary | True | 自动摘要压缩 |
| context_inheritance | True | 上下文继承 |

## 2. 持久记忆模块（L3-R011）

### 2.1 三层记忆结构

| 层级 | 类型 | 存储位置 | 生命周期 |
|------|------|---------|---------|
| L1 | 工作记忆 | 内存 | 会话级 |
| L2 | 情景记忆 | 本地文件 | 项目级 |
| L3 | 语义记忆 | 知识库 | 全局级 |

### 2.2 记忆操作

| 操作 | 描述 | 实现 |
|------|------|------|
| 存储 | 将信息写入记忆 | 文件系统/数据库 |
| 检索 | 根据关键词查找 | 全文搜索/向量匹配 |
| 更新 | 修改现有记忆 | 版本控制 |
| 删除 | 移除记忆 | 软删除机制 |

### 2.3 知识图谱

```python
class KnowledgeGraph:
    def __init__(self): self.nodes, self.edges = {}, []
    def add_node(self, id, type, props): self.nodes[id] = {"type": type, "properties": props}
    def add_edge(self, src, tgt, rel): self.edges.append({"source": src, "target": tgt, "relation": rel})
```

| 节点类型 | 说明 | 关系类型 | 说明 |
|---------|------|---------|------|
| concept | 概念 | has_capability | 拥有能力 |
| agent | 智能体 | requires | 需要 |
| skill | 技能 | similar_to | 相似 |
| task | 任务 | | |

## 3. AI自动记忆行为（L3-R012）

### 3.1 记忆处理流程

```
用户消息 → 【自动记忆处理】（静默） → 回复用户
```

### 3.2 自动记忆处理步骤

#### 第一步：暗知识识别（自动）

| 暗知识特征 | 示例 | 处理方式 |
|-----------|------|---------|
| 个人偏好 | "我习惯..."、"我喜欢..." | 记忆 |
| 团队规则 | "我们团队禁止..." | 记忆 |
| 未来指令 | "以后都这样..." | 记忆 |
| 设计原因 | "因为要兼容..." | 记忆 |

#### 第二步：自动分类

```
包含"好"、"认可" → feedback/
包含文档路径 → reference/
包含项目、设计 → project/
其他个人偏好 → user/
```

## 5. 对话历史管理（L3-R014）

### 5.1 淘汰机制

| 策略 | 触发条件 | 处理 |
|------|----------|------|
| 时间淘汰 | 超过30天 | 自动删除 |
| 空间淘汰 | 超过100MB | LRU淘汰 |
| 重要性淘汰 | 评分低于阈值 | 标记删除 |

### 5.2 历史记录格式

```json
{
  "conversation_id": "uuid",
  "turn": 1,
  "messages": [
    {"role": "user", "content": "...", "timestamp": "..."},
    {"role": "assistant", "content": "...", "timestamp": "..."}
  ],
  "metadata": {
    "token_count": 1000,
    "importance_score": 0.8
  }
}
```

## 8. 诚信原则（L3-R017）

### 8.1 核心原则

| 原则 | 说明 |
|------|------|
| 诚实 | 不伪造、不夸大、不隐瞒 |
| 透明 | 决策过程可追溯 |
| 可问责 | 错误必须承认和修复 |

### 8.2 违规处理

| 违规类型 | 处理方式 |
|---------|---------|
| 轻微违规 | 警告+记录 |
| 中度违规 | 暂停权限+审查 |
| 严重违规 | 立即停止+回滚 |
