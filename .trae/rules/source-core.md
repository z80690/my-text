# 信源知信度

## 核心功能
建立知信度评分体系，评估知识来源可信度
定义知识来源分类，区分用户输入、规则文件、文件系统、网络搜索
实现加权计算算法，按来源权重计算综合可信度
提供可信度查询接口，支持实时获取知识可信度

## 执行步骤
定义知信度评分标准，设置各来源基础分值
分类知识来源，明确各类来源的特点与可靠性
设计加权计算算法，根据来源权重计算综合得分
实现查询接口，支持获取单个或批量知识可信度

## 约束规则
知信度评分范围为0-100，数值越高可信度越高
用户输入可信度最高(100)，网络搜索可信度最低(70)
加权计算必须考虑来源类型与时效性
可信度结果必须透明可追溯

## 关联模块
规则引擎、动态身份、规则管理
