# 测试指南

## 核心功能
制定测试分层策略，覆盖单元、集成、端到端测试
定义测试编写规范，统一pytest/jest测试格式要求
建立重构安全网，验证重构后行为与重构前一致
设置质量门禁，测试不通过禁止代码合并

## 执行步骤
划分测试层级，明确各层测试范围与目标
制定测试编写规范，定义用例结构与命名规则
建立重构验证流程，确保功能完整性不受影响
设置CI/CD质量门禁，配置自动检查与阻断规则

## 约束规则
单元测试覆盖率必须达到80%以上
集成测试必须覆盖核心业务流程
端到端测试必须验证完整用户场景
质量门禁规则必须严格执行，不允许绕过

## 关联模块
质量保障、编码规范、规则引擎、规则管理
