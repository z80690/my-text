# Auto-Refactor Skill

## 功能
全局代码重构：批量安全重构整个项目，保持功能不变

## 使用方式
```
你: 重构所有Class组件为Hooks
你: 统一所有接口地址
你: 批量修改变量命名风格
```

## 重构场景

| 场景 | 说明 | 示例 |
|------|------|------|
| 语法迁移 | 旧语法转新语法 | Class→Hooks, callback→async/await |
| 命名统一 | 统一命名风格 | snake_case→camelCase |
| 模式替换 | 设计模式优化 | 观察者→事件总线 |
| 结构重组 | 目录结构优化 | 扁平→分层 |
| 冗余清理 | 删除重复代码 | 提取公共函数 |

## 执行流程

### 1. 扫描分析
- 全局搜索目标代码
- 分析影响范围
- 制定修改计划

### 2. 安全重构
- 创建备份点
- 逐文件修改
- 保持引用一致

### 3. 验证保证
- 运行测试确保功能不变
- 检查语法错误
- 验证边界情况

## 输出格式
```
【重构计划】
范围: 23个文件
修改: 156处
预计风险: 低

【执行结果】
成功: ✅ 23/23
测试: ✅ 通过
备份: ✅ 已创建
```
