# Auto-Debug Skill

## 功能
智能调试：分析错误、定位问题、修复代码、验证修复

## 使用方式
```
你: 帮我调试这个错误 [粘贴错误信息]
```

## 执行流程

### 1. 错误分析
- 解析错误类型和堆栈信息
- 提取关键错误位置和变量值
- 判断错误分类：语法/逻辑/环境/依赖/并发

### 2. 定位根源
- 在项目中搜索相关代码
- 分析调用链路和数据流
- 找出真正的问题点

### 3. 修复方案
- 生成修复代码
- 确保不影响其他功能
- 添加异常处理

### 4. 验证测试
- 运行测试用例
- 验证修复有效
- 确保无新问题

## 输出格式
```
【错误分析】
类型: ...
位置: ...
原因: ...

【修复方案】
代码改动: ...

【验证结果】
测试通过: ✅/❌
```
