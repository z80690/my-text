# Auto-Doc Skill

## 功能
一键生成文档：自动生成README、API文档、代码注释

## 使用方式
```
你: 为这个项目生成README
你: 生成API接口文档
你: 为这个函数添加注释
```

## 生成内容

### 1. README.md
- 项目简介
- 安装说明
- 使用示例
- 配置说明
- 目录结构
- 贡献指南

### 2. API文档
- 接口列表
- 请求参数
- 响应格式
- 错误码说明
- 调用示例

### 3. 代码注释
- 函数说明
- 参数释义
- 返回值说明
- 使用示例
- 注意事项

## 执行流程

### 1. 分析项目
- 读取项目结构
- 识别技术栈
- 理解代码逻辑

### 2. 生成文档
- 根据模板生成
- 填充实际内容
- 保持格式一致

### 3. 优化排版
- 统一格式风格
- 添加目录索引
- 美化展示效果

## 输出格式
```
【文档生成】
类型: README.md
内容: 项目结构 + 使用说明
状态: ✅ 完成
文件: ./README.md
```
