name: "auto-memory"
description: "自动记忆处理 - 收到用户消息后自动识别暗知识并写入记忆文件，静默执行"
author: "Auto Memory System"
version: "1.0"

## 触发条件
用户发送任何消息时自动触发

## 执行逻辑
1. 分析消息内容，识别是否为暗知识
2. 根据内容分类到对应类型
3. 自动写入 .trae/memories/{类型}/{标题}.md
4. 静默执行，不告知用户

## 暗知识分类规则

### user/ - 用户记忆
- 个人偏好：我习惯、我喜欢、我不喜欢
- 个人背景：我是XX工程师、我主要用XX

### feedback/ - 反馈记忆
- 用户认可：很好、认可、以后都这样
- 用户建议：你应该、建议、改进

### project/ - 项目记忆
- 团队规则：禁止、必须、我们团队
- 设计原因：因为要、为了兼容、历史遗留
- 项目背景：这是XX系统、项目启动时间

### reference/ - 参考记忆
- 外部引用：Jira票号、文档路径、链接

## 明知识（不记忆）
- 版本号：React 18.2.0、Node 18
- 文件位置：在utils.js里、src/utils/
- 函数/变量名：formatDate、config对象

## 文件格式
---
type: {类型}
created: {日期时间}
---

# {标题}

{用户原话}

**Why:** 自动识别的暗知识类型
**How to apply:** 根据类型自动触发

## 静默模式
- 不显示"已保存"提示
- 不打扰用户正常对话
- 完全透明执行
