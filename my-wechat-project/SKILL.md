# 微信小程序云端一体化项目生成器

## 技能目标
本技能的最终目标是帮助用户快速生成一个结构完整、可部署的微信小程序项目。通过理解用户的业务需求，自动生成符合现代开发规范的前后端代码、配置文件和部署脚本。

## 工作流程
1.  **理解业务需求**：明确目标用户、核心场景和功能列表。
2.  **确定技术方案**：根据业务需求选择技术栈和架构。
3.  **生成项目代码**：自动生成前后端代码、配置和部署脚本。
4.  **提供部署指南**：生成详细的部署和运行说明文档。

## 知识库与智能模块调用指南
本技能采用“渐进式披露”架构，集成了多个专业知识模块。我会在需要时按需加载，以提供最精准的协助。

### 核心知识库
*   **业务全景**：关于项目愿景、用户画像、核心功能与业务规则的完整说明，请参阅 [`business-context.md`](./business-context.md) 文件。
*   **技术蓝图**：关于技术栈、项目结构、API规范与部署配置的详细约束，请参阅 [`technical-spec.md`](./technical-spec.md) 文件。

### 领域智能处理模块
当任务涉及以下特定复杂业务时，我将调用对应的**智能处理模块**。该模块内置于知识文件中，可指导我使用专业框架进行自动化分析与决策。

| 涉及领域 | 将加载的智能模块 | 模块核心能力 |
| :--- | :--- | :--- |
| **订单与交易** | [`orders-table.md`](./orders-table.md) | 提供**订单生命周期管理、退款决策、数据分析**的自动化处理框架。 |
| **用户与会员体系** | [`profiles-table.md`](./profiles-table.md) | 提供**用户增长策略、等级与积分设计、促活方案**的自动化处理框架。 |

## 如何使用
开发者可以通过自然语言指令触发此技能。根据您的问题，我将自动调度上述知识库与智能模块。

**通用项目生成指令示例**：
- “生成一个微信小程序电商项目”
- “创建一个包含用户管理和内容发布功能的小程序”

**触发领域智能模块的指令示例**：
- “**帮我分析一下上周的订单取消原因**” → (我将加载 `orders-table.md` 并运用其智能框架进行分析)
- “**为黄金会员设计一个复购激励方案**” → (我将加载 `profiles-table.md` 并运用其智能框架进行设计)
- “**我们平台的订单退款率有点高，帮忙看看问题出在哪，并给出优化建议**” → (我将结合 `business-context.md` 的业务规则与 `orders-table.md` 的分析框架，给出综合报告)

技能将按上述工作流程执行，并最终交付完整的项目代码、部署指南或深度分析报告。
