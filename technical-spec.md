# 项目技术规范与实现约束

## 技术栈
### **前端**
- **框架**: Taro 3.x + React 18
- **语言**: TypeScript
- **构建工具**: Webpack 5
- **CSS处理器**: Less / Sass

### **后端**
- **运行时**: Node.js 18 LTS
- **框架**: Express.js
- **语言**: TypeScript
- **数据库**: MySQL 8.0 (主库), Redis 7.0 (缓存/会话)
- **ORM工具**: TypeORM 或 Prisma (用于提升数据库操作安全性与开发效率)

### **云端平台**
- **计算**: 腾讯云云函数(SCF) - 按需计费，弹性伸缩
- **存储**: 腾讯云对象存储(COS) - 用于图片、文件等静态资源
- **数据库**: 腾讯云云数据库MySQL(CDB) - 高可用，自动备份
- **其他**: 云开发(TCB)、内容分发网络(CDN)
