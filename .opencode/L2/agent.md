## 1. 三层双向同步规则（L2-B001.1）

### 1.1 同步机制

本项目采用**三层双向同步机制**：

| 层级 | 路径 | 角色 |
|------|------|------|
| L1 | `.opencode/L1/agent.md` | 顶层规范/契约 |
| L2 | `.opencode/L2/agent.md` | 项目配置/约束 |
| L3 | `.opencode/rules/*` | 具体实现/规则 |

### 1.2 同步方向

- **向下同步**: L1 → L2 → L3
- **向上同步**: L3 → L2 → L1

### 1.3 同步优先级

| 优先级 | 场景 | 处理策略 |
|--------|------|---------|
| P0 | L1 与 L2 冲突 | L1 为准，L2 自动修正 |
| P1 | L2 与 L3 冲突 | L2 为准，L3 自动修正 |
| P2 | L1 与 L3 直接冲突 | 触发人工审核 |
