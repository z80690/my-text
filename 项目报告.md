## 五、快速启动指南

### 5.1 环境准备

```powershell
# 进入项目目录
cd C:\Users\Administrator\Desktop\my-text\knowledge_graph_project

# 安装Python依赖
pip install playwright pillow loguru lxml psycopg2-binary neo4j networkx pytest pytest-asyncio pytest-cov

# 安装Playwright浏览器
playwright install chromium
```

### 5.2 配置环境变量

编辑 `.env` 文件，填写以下配置：

```bash
# Supabase 配置 (PostgreSQL)
SUPABASE_URL=https://your-project.supabase.co
SUPABASE_KEY=your-anon-key

# 加密配置
ENCRYPTION_KEY=your-encryption-key-32-chars

# 应用配置
HOST=0.0.0.0
PORT=8080
```

### 5.3 启动服务

```powershell
# 启动FastAPI服务
python main.py
```

### 5.4 访问地址

| 服务 | 地址 |
|------|------|
| API服务 | http://localhost:8080 |
| API文档 | http://localhost:8080/docs |
| 健康检查 | http://localhost:8080/health |
