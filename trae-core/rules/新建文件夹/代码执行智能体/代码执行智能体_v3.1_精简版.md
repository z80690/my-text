## 第四章：博弈调度机制

### 4.1 三种模式

| 模式 | 触发关键词 | 工作流 |
|-----|---------|-------|
| 模式一：辩论 | 对比、权衡、优缺点 | 视角A → 视角B → 裁决 |
| 模式二：降维打击 | 优化、改进、重构 | 生成 → 挑剔 → 融合 |
| 模式三：深度设计 | 设计、架构、方案 | 蓝图 → 挑战 → 融合 |

### 4.2 触发决策流程

```
任务 → 分词 → query_kg查询 → 概念聚合
    │
    ├── 含"辩论" → 模式一
    ├── 含"优化" → 模式二
    ├── 仅"设计" → 模式三
    └── 无概念 → 复杂度判断 → 模式三/常规路由
```

### 4.3 执行流程

**模式一（辩论）**：
1. 识别对立视角（创新探索 vs 风险控制）
2. 视角A输出结论
3. 视角B提出反对意见
4. 综合裁决：共识、分歧、建议

**模式二（降维打击）**：
1. 创意智能体生成方案初稿
2. 审查智能体提出审阅意见
3. 综合智能体融合优化

**模式三（深度设计）**：
1. 架构师智能体输出设计蓝图
2. 工程师智能体评估可行性
3. 技术作家输出落地方案

## 第七章：上下文感知路由

### 7.1 路由策略

| 策略 | 优先级 | 说明 |
|-----|-------|------|
| 语义路由 | 最高 | 知识图谱意图分析 |
| 上下文路由 | 高 | 历史任务+用户偏好 |
| 负载路由 | 中 | 选择负载最低智能体 |
| 优先级路由 | 中 | 高优先级任务优先 |

### 7.2 路由算法

```python
def select_agent(task, context):
    if context.get("context_id"):
        similar = get_context(context["context_id"]).query_similar_tasks(task)
        if similar and similar.agents_used:
            return similar.agents_used[0]
    if context.get("user_preference"):
        preferred = context["user_preference"].get("agent")
        if preferred and is_available(preferred):
            return preferred
    recommended = kg.recommend_agents(task)
    candidates = [a for a in agents if a.id in recommended]
    return min(candidates, key=lambda a: a.current_load)
```

### 7.3 上下文复用策略

| 场景 | 策略 |
|-----|------|
| 连续同类任务 | 复用智能体组合 |
| 用户明确偏好 | 优先使用偏好智能体 |
| 新类型任务 | 语义路由+记录历史 |
| 任务失败 | 切换备用智能体 |

## 第八章：执行流程

```
用户输入 → Step1:接收任务 → Step2:图谱查询+上下文分析
                                │
                                ▼
           Step3:上下文感知路由 → Step4:智能体调度 → Step5:结果聚合+更新
                                │
                                ▼
                         Step6:返回结果
```

**Step1**：解析输入、关联/创建上下文、检查相似任务  
**Step2**：分词查询、分析偏好历史、确定博弈模式  
**Step3**：优先缓存、语义路由+负载均衡  
**Step4**：总线分发、记录历史  
**Step5**：收集输出、更新记忆  
**Step6**：输出结果、记录日志

### 8.1 响应格式

```json
{
  "status": "success",
  "agent_id": "code_executor_agent",
  "mode": "game_theory_mode2",
  "task": "...",
  "context_id": "ctx_xxx",
  "result": {},
  "agents_used": [],
  "context_updated": true,
  "execution_time_ms": 1250
}
```

### 8.2 错误处理

| 错误码 | 说明 | 处理 |
|-------|------|-----|
| ERR_AGENT_UNAVAILABLE | 智能体不可用 | 切换备用 |
| ERR_CONTEXT_EXPIRED | 上下文过期 | 重新创建 |
| ERR_STATE_LOCKED | 状态锁定 | 等待/解锁 |
| ERR_ROUTE_FAILED | 路由失败 | 检查总线 |
| ERR_TIMEOUT | 执行超时 | 增加超时 |
