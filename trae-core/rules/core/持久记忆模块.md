# 持久记忆模块 v1.0

## 文档说明

本文档为持久记忆模块实现规则，定义跨会话的项目级知识存储机制。

---

## 第一章：模块概述

### 1.1 基本信息

| 属性 | 值 |
|-----|-----|
| 中文名 | 持久记忆模块 |
| 英文名 | Persistent Memory Module |
| ID | L3-M-PERSISTENT |
| 版本 | v1.0 |
| 所属层级 | L3 |

### 1.2 核心能力

| 能力 | 说明 |
|-----|------|
| 长期存储 | 跨会话持久化存储 |
| 结构化记忆 | Markdown + YAML Frontmatter格式 |
| 版本控制 | Git兼容的变更追踪 |

---

## 第二章：记忆文件格式

### 2.1 标准格式

```markdown
---
type: user
created: 2026-05-09
---

# 记忆名称

规则内容。

**Why:** 决策理由
**How to apply:** 触发场景
```

## 第三章：四种记忆类型

### 3.1 目录结构

```
.trae/memories/
├── user/           # 用户记忆
├── feedback/       # 反馈记忆
├── project/        # 项目记忆
└── reference/      # 参考记忆
```
