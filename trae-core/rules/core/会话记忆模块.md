# 会话记忆模块 v1.0

## 文档说明

本文档为会话记忆模块实现规则，定义当前对话中的临时上下文管理机制。

---

## 第一章：模块概述

### 1.1 基本信息

| 属性 | 值 |
|-----|-----|
| 中文名 | 会话记忆模块 |
| 英文名 | Session Memory Module |
| ID | L3-M-SESSION |
| 版本 | v1.0 |
| 所属层级 | L3 |

### 1.2 核心能力

| 能力 | 说明 |
|-----|------|
| 上下文存储 | 当前对话的所有消息和工具调用 |
| TTL管理 | 自动过期清理 |
| 窗口限制 | 上下文窗口管理 |

---

## 第二章：数据结构

### 2.1 会话上下文

```python
@dataclass
class SessionContext:
    conversation_id: str
    turn: int
    messages: List[Message]
    created_at: datetime
    last_accessed: datetime
    ttl: int  # 秒
```

### 2.2 消息结构

```python
@dataclass
class Message:
    role: str  # "user" | "assistant" | "system"
    content: str
    timestamp: datetime
```

## 第三章：核心功能

### 3.1 TTL管理

```python
class SessionMemory:
    def __init__(self, ttl: int = 86400):
        self.ttl = ttl
        self.sessions: Dict[str, SessionContext] = {}
```
