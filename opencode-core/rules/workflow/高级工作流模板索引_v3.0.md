# 高级工作流模板索引 v3.0

## 概述

本文档为智能体团队调度员 v3.0 的可扩展工作流引擎提供完整的模板索引。

---

## 模板索引

| 模板ID | 模式类型 | 核心功能 |
|-------|---------|---------|
| seq_flow | 顺序流 | 按顺序依次执行多个任务节点 |
| parallel_gateway | 并行网关 | 多个分支任务同时执行 |
| conditional_branch | 条件分支 | 根据条件表达式选择执行路径 |
| loop_flow | 循环 | 重复执行任务直到满足退出条件 |
| split_join | 分合流 | 将任务拆分为多个子任务并行执行 |
| compensation | 补偿事务 | 当流程失败时自动执行补偿/回滚操作 |
| game_theory | 博弈模式 | 多智能体辩论决策模式 |
| monitor_flow | 监控流 | 全程监控任务执行状态 |

---

## 模板详细定义

### 1. 顺序流 (seq_flow)

```yaml
template_id: seq_flow
name: 顺序流
description: 按顺序依次执行多个任务节点
qos_level: normal

execution:
  mode: sequential
  nodes:
    - task_ref: task_1
      next: task_2
```

### 2. 并行网关 (parallel_gateway)

```yaml
template_id: parallel_gateway
name: 并行网关
description: 多个分支任务同时执行
```
