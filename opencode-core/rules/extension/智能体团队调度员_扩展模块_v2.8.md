## 第一章 架构设计详解

### 1.1 五层架构

```
┌─────────────────┐
│ 中央调度层      │ 路由│均衡│聚合│状态
├─────────────────┤
│ 监控层          │ 执行│性能│异常│日志
├─────────────────┤
│ 上下文管理层    │ 工作│情景│语义记忆
├─────────────────┤
│ 知识图谱层      │ 概念│关系│意图推断
├─────────────────┤
│ 智能体层        │ 23专业智能体
└─────────────────┘
```

### 1.2 三层记忆

| 类型 | 内容 | 周期 | 用途 |
|-----|-----|-----|-----|
| 工作记忆 | 当前对话、临时状态 | 24h | 实时上下文 |
| 情景记忆 | 任务历史、执行记录 | 30d | 历史查询、路由优化 |
| 语义记忆 | 领域知识、术语定义 | 持久 | 长期知识存储 |
