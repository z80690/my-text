## 代码风格

### 命名规范
- 变量：camelCase（userName, isActive）
- 类：PascalCase（UserService, ConfigLoader）
- 常量：UPPER_SNAKE_CASE（MAX_RETRY, API_KEY）
- 文件：kebab-case 或 snake_case（user-service.py, user_service.py）

### 必须遵循
- 缩进：4空格
- 类型提示：所有函数必须标注类型
- 导入顺序：标准库 → 第三方库 → 本地模块

### 禁止清单
- 禁止 var、console.log、硬编码值
- 禁止空 import 或 pass 占位
- 禁止一次性代码（无复用性）
