## 三、技术架构

### 3.1 多Agent协作体系

Marvis内置6个Agent组成的"AI团队"，7×24小时待命：

| Agent名称 | 职责 | 核心能力 |
|-----------|------|----------|
| **主Agent** | 项目经理 | 理解需求、拆解任务、分配调度 |
| **File Agent** | 文件管家 | 文件搜索、内容理解、格式转换、分类归档 |
| **Computer Agent** | 系统运维 | 硬件检测、系统设置、性能优化、垃圾清理 |
| **App Agent** | 应用操控 | 打开/关闭软件、界面交互、自动化流程 |
| **Browser Agent** | 网页交互 | 网页浏览、数据抓取、表单填写 |
| **Search Agent** | 信息检索 | 联网搜索、资料整理、摘要生成 |

### 3.2 虚拟办公室界面

用户可以在虚拟办公室里实时查看每个Agent的状态：
- **工作中**：坐在工位认真"搬砖"
- **空闲时**：打盹、喝咖啡、闲逛、健身、上厕所

这种可视化设计增加了产品的趣味性和亲和力。

### 3.3 底层技术支撑

| 技术合作方 | 合作内容 |
|-----------|----------|
| **微软** | WinML推理框架、MCP on Windows系统级文件服务 |
| **英特尔** | OpenVINO加速技术，芯片级运算速度优化（提升20%+） |
| **腾讯自研** | Qwen端侧模型（隐私模式使用） |

## 四、运行模式

### 4.1 双模式设计

| 模式 | 特点 | 适用场景 |
|------|------|----------|
| **效率模式** | 端云协同，复杂任务借助云端模型（腾讯混元 + DeepSeek V4） | 日常办公、普通任务 |
| **隐私模式** | 完全离线，基于Qwen端侧模型，数据不出设备 | 财务、法务、HR等敏感场景 |

### 4.2 安全机制

- **L2级安全兜底机制**：删除文件、修改系统配置等敏感操作需用户确认
- **支付动作**：必须由用户本人完成
- **数据保护**：隐私模式下所有数据解析、图片识别及对话均在本地完成

## 五、支持平台与系统要求

### 5.1 支持平台

| 平台 | 状态 |
|------|------|
| **Windows 10/11** | ✅ 已开放 |
| **macOS** | ✅ 已开放 |
| **Android** | ✅ 已开放 |
| **iOS** | ⏳ 开发中（预计2026年6月中旬） |

### 5.2 系统要求

| 配置 | 最低要求 | 推荐配置 |
|------|----------|----------|
| **CPU** | 8核（Intel Core i7或AMD Ryzen 7） | 8核及以上 |
| **内存** | 16GB RAM | 16GB RAM或更高 |
| **存储** | SSD，2GB可用空间 | NVMe SSD |
| **显卡** | - | 独立显卡（图像处理更佳） |
| **网络** | 互联网连接（隐私模式可离线） | 稳定网络 |

**为什么需要8核CPU？**  
Marvis采用1主Agent + 5副Agent的协作架构，多核心能提供更强的并行处理能力。隐私模式下依赖本地端侧模型推理，对CPU性能要求更高。

## 七、产品对比

### 7.1 与传统AI助手对比

| 维度 | 传统AI助手（如豆包、Kimi） | Marvis |
|------|--------------------------|--------|
| **定位** | 通用对话型 | 操作系统层级 |
| **权限** | 应用层（挂在系统上） | 系统层（长在系统里） |
| **文件操作** | 需手动上传 | 直接访问本地文件 |
| **系统控制** | 有限（模拟点击） | 深度操控（底层API） |
| **跨端能力** | 弱 | 强（手机控电脑、电脑控手机） |

### 7.2 与同类产品对比

| 产品 | 特点 | 对比 |
|------|------|------|
| **微软Copilot** | 深度整合Windows系统 | 功能类似，但Marvis支持跨端协同 |

## 十、总结

### 10.1 核心亮点

| 亮点 | 说明 |
|------|------|
| **系统级权限** | 直接穿透操作系统底层，真正实现"一句话操控电脑" |
| **多Agent协作** | 6个AI智能体分工协作，像一个团队在工作 |
| **跨端协同** | 手机控电脑、电脑控手机，实现无缝衔接 |
| **隐私保护** | 隐私模式数据不上云，满足敏感场景需求 |
| **免费额度充足** | 每天1000万Token，日常使用完全免费 |

### 10.2 适用场景

- ✅ 电脑系统设置与优化
- ✅ 文件管理与格式转换
- ✅ 办公文档处理与数据分析
- ✅ 跨设备远程控制
- ✅ 隐私敏感的数据处理

### 10.3 潜在挑战

| 挑战 | 说明 |
|------|------|
| **硬件要求较高** | 需8核CPU、16GB内存，老设备可能无法运行 |
| **iOS版本未上线** | iPhone用户暂无法使用 |
| **生态整合** | 需要更多软件厂商授权，扩展能力有限 |
